## Version Control

###  Git Basics
- **Staging** → Preparing files for commit  
- **Commit** → Saving changes  
- **Push** → Uploading to remote repo  
- **Pull** → Fetching updates  
- **Merge** → Combining changes  

###  Branching Strategy
- `main` → Stable code  
- `dev` → Development  
- `feature/*` → New features
